Model Prediction for dataset on Data Science

The project has as objective the prediction of activity of chemical compounds from the given dataset: An analysis of a model prediction is given from a dataset. The complete dataset consists in two sets, a training and test sets, while for the training set, this is separated into validation and training sets. The order and methodology consists in the collection of features, data preparation, models identification, model selection, test given the different selected models and presentation of the results.

For the modelling part, considered the most important one, was done given the stratified k-fold technique once the training set was given, and the optimization of the hyperparameters for each model was done thanks to GridSearchCV. This function pairs the input hyper-parameters for classifier, generates and learns the input data separated by the given cross validation method. Later on, based on the AUC score, for each paired parameter, returns the model and the best parameters with the highest performance. Then, according to the comparison between the predicted output and the ACTIVE column for the validation set, the ROC_AUC_SCORE, we use it to calculate the AUC score.

The models used for the predition and testing of the dataset are the Random Forest Classifier (RFC), Multilayer Perceptron Classifier (MLP), Multinomial Naive Bayes and BernoulliNB Naive Bayes. From them different results for the basic information datasets and fingerprint dataset were obtained, finding that the one giving the best AUC score is the MLP model while using the dataset containing both the original and the fingerprints information.
